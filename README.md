# Projeto-Hospitalar
O sistema pode ser acessado diretamente pelo navegador: 
https://murillojava.github.io/Projeto-Hospitalar/

Ou localmente, seguindo esses passos:

Clone o repositório - git clone https://github.com/murilloJAVA/Projeto-Hospitalar.git ; 

cd Projeto-Hospitalar ;

Abra o arquivo index.html ;

*Não é necessário instalar dependências, pois o projeto utiliza apenas HTML, CSS e JavaScript puro.


Plataforma utilizada: GitHub Pages


Funcionalidades: 

  Página de Cadastro: É onde o usuário pode cadastrar a missão que o robô realizará, indicando algumas informações, como ponto de origem, destino, nível de prioridade, entre outras.
  
  Lista: Página em que o usuário pode visualizar a lista de missões, com todas as informações, que deverá ser feita pelo robô. Também possui o controle sobre começar, indicar falha, reagendar e até mesmo cancelar uma missão. 
  
  Telemetria: Acompanhamento do robô em tempo real. É possível verificar bateria, latência e o status. Quando alguma missão se concretiza ou apresenta uma falha, a informação é exibida nessa página, com um Json.
